The Foundation Services Scheduler is built using the TopShelf framework.

TopShelf documentation: https://app.readthedocs.org/projects/topshelf/downloads/pdf/stable/


Usage:
Foundation.Services.Scheduler install

Foundation.Services.Scheduler.exe install -servicename "ABC1" -displayname "ABC1"

Foundation.Services.Scheduler.exe install -username:DOMAIN\ServiceAccount -password:itsASecret -servicename:AwesomeService –autostart
